# MontreHack website
Quick how-to.

## Local testing

* Install dependencies:

        $ sudo apt-get install jekyll ruby-redcarpet

  Could also be done from `gem`

* Run the dev server:

        $ jekyll --server --auto

* View the site: http://localhost:4000/
