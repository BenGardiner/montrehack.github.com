# MontreHack website
Quick how-to.

## Local testing

* Install jekyll: http://jekyllrb.com/docs/installation/ (don't use packaged versions they are too old)

* Run the dev server:

        $ jekyll serve --watch

* View the site: http://localhost:4000/
